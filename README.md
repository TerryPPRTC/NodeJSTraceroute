# NodeJSTraceroute

## To build

Use node-gyp command.  The node-gyp need to be installed by npm.
```
npm install -g node-gyp
```
Then use node-gyp to build the project. Try these commands:
```
node-gyp configure
node-gyp generate
node-gyp build
node-gyp rebuild 
```
At last call the sample index.js:
```
node index.js
```
